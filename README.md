# Как запустить

Для запуска требуется установленный Python версии 3.6 и выше

- Скачайте код
- Установите зависимости из requirements.txt
```
pip install -r requirements.txt
```
- Перейти в папку /tests  ```cd tests``` 
- Запустите программу:
```
pytest elemenets_test.py
```
